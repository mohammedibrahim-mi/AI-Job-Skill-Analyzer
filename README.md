# AI Job Skill Analyzer
AI agent that analyzes job roles, finds your skill gaps from live job postings and builds a 90-day learning roadmap and produces a job readiness PDF report automatically.

Steps to run: 

1.Add Open Router API and SERPER API secret key in .env file.
2.Run command: 

pip install crewai-tools   
pip install reportlab  
python main.py
